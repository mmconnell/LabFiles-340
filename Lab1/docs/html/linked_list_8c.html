LinkedList* linkedList ( )

The so called "constructor" for the linked list.

The linked list constructor builds a non circular linked list that contains a dummy head not. This is meant to be similar to a Java linked list constructor.

Returns
LinkedList * - The linked list pointer that contains a dummy head node but is not circular.
