int countRecords ( FILE *  fin,
int  linesPerRecord 
)

Counts the number of records that are in the file.

Counts the number of lines in the file and then divides by the number of lines per record.

Note
The FILE * is rewound at the end of the function
Parameters
fin- The FILE * representing the open file
linesPerRecord- Representing the number of lines in the file that make up a record
Returns
int - Representing the number of records in the file
Warning
- The passed in FILE * fin is checked - exit(-99) if NULL
- The passed in int linesPerRecord is checked - exit(-99) if <= 0
- If count is 0 the program is ended with a -99 status
FILE* openInputFile_Prompt ( )

Prompts the user for the name of the input file and attempts to open that file.

Prompts the user for the name of the input file Trys to open the file, if the file is opened the FILE * is returned If the file does not open, the user is reprompted.

Note
The carriage return is removed from the input buffer
Returns
FILE * - Representing the open file
